# OS Scheduler
Creating Processes and attach time for each process then Apply Scheduling Algorithms.

1. First Come First Serve (FCFS).

2. Higher Priority First (HPF).

3. Shortest Remaining Time Next (SRTN).

4. Round Robin (RR).
